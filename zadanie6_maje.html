Zadanie 6 – Teoria JavaScript

1) Rodzaje okienek
W JavaScript wyróżniamy następujące rodzaje okienek dialogowych:
Okienko alert – wyświetla okienko dialogowe z informacją i przyciskiem OK.
Okienko prompt – generuje okno, w którym użytkownik może wpisać wartość, zwracaną następnie do programu.
Okienko confirm – służy do tego, aby użytkownik coś potwierdził (stąd nazwa confirm).

2) Konwersji liczb
Wczytanie zmiennej z użyciem prompt wykonywane jest jako ciąg znaków.
Dlatego konieczna jest konwersja z ciągu znaków na wartości liczbowe, aby wykonywać obliczenia:
parseFloat(zmienna) – konwersja zmiennej do typu float
parseInt(zmienna) – konwersja zmiennej do typu int (liczbowego)
toString(zmienna) – konwersja zmiennej do typu łańcuchowego

3) Przy nadawaniu nazw zmiennym należy przestrzegać pewnych zasad:
1. Nazwa zmiennej musi się zaczynać od litery, znaku $ lub _, może zawierać tylko litery (bez polskich znaków), cyfry oraz $ i _.
Poprawne nazwy zmiennych: zmienna1, $wynik, _rozmiar, wynik1, Rekord, $_gracz.
Błędne nazwy zmiennych: 1zmienna, wynik gry, wysokosc&szerokosc, wynik.rekord, &gracz, wysokość, imie+nazwisko.

2. Wielkość liter ma znaczenie.
Rekord i rekord to dwie różne zmienne!

3. Należy nie używać słów kluczowych JavaScript jako nazw zmiennych.

4) Typy zmiennych w JavaScript
W JavaScript nie ma możliwości definiowania różnych typów zmiennych tak jak w C++.
Typy zmiennych są definiowane w trakcie ich przypisywania.

Podstawowe typy zmiennych:
Number – typ liczbowy (zmienne całkowite i zmiennoprzecinkowe)
String – typ tekstowy
Boolean – typ logiczny
Null – pusta zawartość
Undefined – typ niezidentyfikowany
Object – obiekt
